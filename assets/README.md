# 图标文件说明

请在此目录放置以下图标文件：

## 必需文件

1. **icon.png** (1024x1024 像素)
   - 应用主图标
   - 正方形，PNG 格式
   - 建议使用圆角设计

2. **adaptive-icon.png** (1024x1024 像素)
   - Android 自适应图标
   - 前景图（图标主体）
   - 背景色在 app.json 中配置为 #4CAF50

## 可选文件

3. **splash.png** (1284x2778 像素)
   - 启动画面
   - 如果不提供，将使用纯色背景

4. **favicon.png** (48x48 像素)
   - Web 图标
   - 仅用于 Web 版本

## 快速生成图标

可以使用以下在线工具生成：

- [App Icon Generator](https://www.appicon.co/)
- [Icon Kitchen](https://icon.kitchen/)
- [Favicon Generator](https://www.favicon-generator.org/)

## 设计建议

- 使用绿色主题色 (#4CAF50)
- 简洁设计，避免过多细节
- 可以包含"轻簿"文字或"Q"字母
- 确保在小尺寸下清晰可辨
